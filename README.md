# bookify
Bookify is a web application that allows users to manage a personal library of books.    
Users can add, edit, and delete book entries while also sorting them by title or rating.    
The application integrates a quote API to provide inspirational quotes alongside the book showcase.   

## Features

- **Add Books**: Users can add new book entries with details like title, author, rating, and summary.
- **Edit Books**: Existing book entries can be edited.
- **Delete Books**: Users can delete book entries from the database.
- **Sort Books**: Users can sort the displayed books by title or rating.
- **Random Quotes**: The application fetches a random inspirational quote from an external API.

## Technologies Used

- **Frontend**: HTML, CSS, EJS, Bootstrap, Font Awesome
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **API**: Axios for making HTTP requests
